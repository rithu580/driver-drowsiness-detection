# Driver Drowsiness Detection System

## Overview
A real-time driver monitoring system that detects fatigue using computer vision
and triggers alerts via Android, Firebase, and emergency communication.

## Features
- Eye Aspect Ratio (EAR) based drowsiness detection
- Firebase real-time synchronization
- Android foreground service alarm
- Auto emergency call & SMS with GPS
- Live Google Maps tracking

## Tech Stack
- Python, OpenCV, MediaPipe
- Firebase Realtime Database
- Android (Kotlin, Jetpack Compose)
- Google Maps SDK

## Use Cases
- Commercial vehicles
- Long-distance drivers
- Smart transportation systems
